# Генерация изображений бинго-листов

## Инструкция по использованию

1. Откройте файл `generate-bingo-images.html` в браузере
2. Нажмите кнопку "Сгенерировать все 40 изображений"
3. Браузер начнет автоматически скачивать изображения
4. После завершения генерации переместите все скачанные файлы в папку `bingo-sheets-images`

## Альтернативный способ (Node.js)

Если у вас установлен Node.js, вы можете использовать скрипт для автоматической генерации:

```bash
npm install puppeteer
node generate-images.js
```

## Примечание

Из-за ограничений безопасности браузера, файлы не могут быть сохранены напрямую в папку. Они будут скачаны в папку загрузок, откуда их нужно переместить в `bingo-sheets-images`.
